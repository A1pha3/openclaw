访问控制界面需要认证：

## 🌐 第五部分：远程访问

### 5.1 配置远程访问

#### 方式 1：Tailscale Serve

```json5
{
  gateway: {
    bind: "loopback",
    tailscale: {
      mode: "serve"  // serve | funnel | off
    }
  }
}
```

**Tailscale 模式对比**：

| 模式 | 访问范围 | 安全性 |
|------|----------|--------|
| serve | 仅 tailnet 内 | ⭐⭐⭐⭐⭐ |
| funnel | 公网访问 | ⭐⭐⭐ |
| off | 无 | - |

#### 方式 2：SSH 隧道

```bash
# 本地端口转发
ssh -L 18789:127.0.0.1:18789 user@gateway-host

# 远程访问 http://127.0.0.1:18789/
```

#### 方式 3：反向代理

**Nginx 配置**：

```nginx
server {
    listen 443 ssl;
    server_name openclaw.example.com;
    
    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;
    
    location / {
        proxy_pass http://127.0.0.1:18789;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 5.2 安全考虑

| 配置 | 说明 | 重要性 |
|------|------|--------|
| HTTPS | 必须使用 HTTPS | ⭐⭐⭐⭐⭐ |
| 令牌认证 | 设置网关令牌 | ⭐⭐⭐⭐ |
| IP 白名单 | 限制访问 IP | ⭐⭐⭐ |
| 密码保护 | 设置访问密码 | ⭐⭐⭐ |

## 🔧 第七部分：配置选项

### 7.1 Web 服务配置

```json5
{
  gateway: {
    port: 18789,
    canvasHost: {
      enabled: true,
      port: 18793
    }
  }
}
```

### 7.2 认证配置

```json5
{
  gateway: {
    auth: {
      mode: "token",           // token | password | off
      token: "${GATEWAY_TOKEN}",
      allowTailscale: true,    // 允许 Tailscale 访问
      password: "your-password" // 密码模式
    }
  }
}
```

### 7.3 CORS 配置

```json5
{
  web: {
    cors: {
      enabled: true,
      origins: [
        "http://localhost:3000",
        "https://myapp.example.com"
      ]
    }
  }
}
```

## 🐛 第九部分：故障排除

### 9.1 页面无法访问

```bash
# 检查网关是否运行
openclaw status

# 检查端口
lsof -i :18789

# 检查日志
openclaw logs --lines 50
```

### 9.2 WebSocket 连接失败

```bash
# 测试 WebSocket 连接
curl -i -N -H "Connection: upgrade" \
  -H "Upgrade: websocket" \
  http://127.0.0.1:18789/
```

### 9.3 认证失败

```bash
# 检查令牌配置
openclaw config get gateway.auth

# 重置令牌
openclaw config set gateway.auth.token "$(openssl rand -hex 32)"
```
